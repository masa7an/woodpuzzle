プライバシーポリシー / Privacy Policy

本ゲーム「Wood Puzzle」（以下、当ゲーム）における利用者情報の取り扱いについて、以下のとおり定めます。

This policy describes how "Wood Puzzle" handles user information.

1. 取得する情報 / Information Collection

当ゲームでは、サービスの改善および利用状況の分析のため、以下の情報を自動的に取得する場合があります。

  • アクセス日時 / Access timestamp
  • 利用端末・ブラウザ情報 / Device and browser information
  • ゲーム内の行動データ（ステージクリア、リトライ回数など） / In-game events (Stage clears, retries, etc.)

※これらの情報に、特定の個人を識別できる情報（氏名、電話番号、メールアドレス等）は含まれません。

* This data does not include personally identifiable information.

2. 利用目的 / Purpose of Use

取得した情報は、以下の目的でのみ利用します。

  • ゲームの機能改善および不具合修正 / Improving game features and fixing bugs
  • 利用状況の統計分析 / Statistical analysis of usage

3. 利用ツール / Tools Used

当ゲームでは、アクセス解析のために以下のツールを使用しています。

  • Google Analytics 4 (GA4)

Google Analyticsは、Cookie（クッキー）を使用してデータを収集します。このデータ収集は匿名で行われ、個人を特定するものではありません。

We use Google Analytics 4, which uses cookies to collect anonymous data.

4. お問い合わせ / Contact

本ポリシーに関するお問い合わせは、ゲーム公開元の連絡先（GitHub Issues等）までお願いいたします。
